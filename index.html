Simon Says Game

Objectives : Remember the sequence shown by the computer and repeat it correctly to score points.

How to Play ?

1. The game starts when you press the Start button.

2. The computer will highlight a random color/button.

3. Watch carefully and remember the sequence.

4. After the sequence ends, repeat it by clicking the buttons in the same order.

5. Each round, a new step is added to the sequence, making it harder.

Rules :

Follow the exact order of the sequence.

If you click the wrong button, the game is over.

Try to remember longer sequences to get a higher score.

Stay focused and improve your memory skills.

Press any key to start the game.
